Датчик температуры

В этом эксперименте мы считываем данные с датчика температуры.

Список деталей для эксперимента

  • 1 датчик температуры
  • 5 резисторов номиналом 1 кОм
  • 4 провода

Схема на макетке

Скетч

            #include 
OneWire ds(10);
 
void setup() {
  Serial.begin(9600); 
}
 
void loop() {
  byte data[2];
  ds.reset(); 
  ds.write(0xCC);
  ds.write(0x44);
  delay(750);
  ds.reset();
  ds.write(0xCC);
  ds.write(0xBE);
  data[0] = ds.read(); 
  data[1] = ds.read();
  int Temp = (data[1]<< 8)+data[0];
  Temp = Temp>>4;
  Serial.println(Temp);
}

Обратите внимание

  • Используется дополнительная библиотека OneWire.
  • У датчика температуры довольно сложный интерфейс работы. Прочитайте инструкцию прежде чем работать с ним.
